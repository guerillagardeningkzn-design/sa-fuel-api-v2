# ✦ SA Fuel Price API v2

**MD Works · Project 05 — TypeScript + Cloudflare Workers + D1**  
TypeScript · Cloudflare Workers · D1 (SQLite) · Zod · Wrangler

Monthly South African fuel retail prices as published by the DMRE.
Fully serverless — runs on Cloudflare's global edge. Zero infrastructure cost.

**Live:** `https://sa-fuel-api.workers.dev`  
**Docs:** `https://sa-fuel-api.workers.dev/docs`

---

## What changed from v1

| v1 (Node + Railway)     | v2 (Workers + D1)            |
|-------------------------|------------------------------|
| Express                 | Cloudflare Worker fetch handler |
| PostgreSQL via `pg`     | D1 (SQLite at the edge)      |
| `railway.toml`          | `wrangler.toml`              |
| JavaScript              | TypeScript                   |
| No validation library   | Zod schema validation        |
| `npm start`             | `wrangler deploy`            |

## Local Development

```bash
npm install
wrangler dev       # starts local Worker at localhost:8787
```

For local D1 (SQLite file on disk):
```bash
npm run db:local   # applies schema to local D1
```

## Deployment — Step by Step

### 1 — Install Wrangler

```bash
npm install
npx wrangler login   # opens browser to authenticate with Cloudflare
```

### 2 — Create the D1 database

```bash
npx wrangler d1 create sa-fuel-prices
```

Copy the `database_id` from the output and paste it into `wrangler.toml`:

```toml
[[d1_databases]]
binding       = "DB"
database_name = "sa-fuel-prices"
database_id   = "PASTE_YOUR_ID_HERE"
```

### 3 — Run the schema migration

```bash
npm run db:migrate
```

Creates the `fuel_prices` table, the `update_timestamp` trigger and the month index.

### 4 — Seed 29 months of historical data

```bash
# Generate the seed SQL file
node scripts/generate-seed.js > migrations/0002_seed.sql

# Apply it to D1
npx wrangler d1 execute sa-fuel-prices --file=migrations/0002_seed.sql
```

### 5 — Set the API key secret

```bash
npx wrangler secret put API_KEY
# prompted to enter value — type your key, press Enter
```

This is stored encrypted in Cloudflare. It never appears in code or `wrangler.toml`.

### 6 — Deploy

```bash
npm run deploy
```

Wrangler bundles the TypeScript, deploys to Cloudflare's edge and prints your URL.

### 7 — Verify

```bash
curl https://sa-fuel-api.workers.dev/v1/prices/latest
```

You should see May 2026 prices in the response.

## Adding a new month (first Wednesday each month)

```bash
curl -X POST https://sa-fuel-api.workers.dev/v1/prices \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "month":      "2026-06",
    "monthLabel": "Jun 2026",
    "p95i":  27.19,
    "p95c":  26.32,
    "p93i":  27.08,
    "d005i": 27.93,
    "d005c": 27.05
  }'
```

Or use the admin page in the SA Fuel Tracker (`/admin.html`) — no terminal needed.
